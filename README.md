#### Esse é um sistema desenvolvido utilizando os conceitos de microserviços abrangindo algumas das boas praticas de mercado.

<li> Spring Boot </li>
<li> Spring Open Feing </li>
<li> Java 17 </li>
<li> JUnit5 </li>
<li> Mockito </li>
<li> Z-API </li>
